Programming Basics

Introduction to Programming

Programming is the process of creating a set of instructions that tell a computer how to perform a task. These instructions are called programs, and they are written in special languages called programming languages.

Key Programming Concepts

  • Sequence - Instructions executed in order
  • Selection - Making decisions in code
  • Iteration - Repeating actions
  • Variables - Storing and manipulating data
  • Functions - Reusable blocks of code

Variables and Data Types

Understanding Variables

Variables are containers for storing data values. Think of them as labeled boxes that can hold different types of information.

Variable Declaration Examples:


# Number variables
score <-- 100
temperature <-- 98.6

# Text variables
name <-- "Alice"
message <-- "Hello, World!"

# Boolean variables
is_student <-- 1  # TRUE
has_passed <-- 0  # FALSE

# Lists
numbers <-- [1, 2, 3, 4, 5]
names <-- ["Alice", "Bob", "Charlie"]

Common Data Types

Type Description Example
Integer Whole numbers 42, -17, 1000
Float Decimal numbers 3.14, -0.001, 2.0
String Text data "Hello", 'World'
Boolean True/False values True, False
List Ordered collection [1, 2, 3]

Control Structures

If Statements (Selection)

If statements allow your program to make decisions based on conditions.


# Basic if statement
age <-- 18
IF age >= 18 {
    DISPLAY("You are an adult")
} ELSE {
    DISPLAY("You are a minor")
}

# Multiple conditions
score <-- 85
IF score >= 90 {
    grade <-- "A"
} ELSE IF score >= 80 {
    grade <-- "B"
} ELSE IF score >= 70 {
    grade <-- "C"
} ELSE {
    grade <-- "F"
}

Loops (Iteration)

Loops allow you to repeat a block of code multiple times.

For Loop Example:


# Counting from 1 to 5
REPEAT 5 TIMES {
    DISPLAY(i)
}

# Iterating through a list
fruits <-- ["apple", "banana", "orange"]
length <-- LENGTH(fruits)
i <-- 1
REPEAT length TIMES {
    DISPLAY(fruits[i])
    i <-- i + 1
}

While Loop Example:


# Counting down from 5
count <-- 5
REPEAT UNTIL (count <= 0) {
    DISPLAY(count)
    count <-- count - 1
}

For vs While Loops

For Loop While Loop
Used when number of iterations is known Used when number of iterations is unknown
Iterates over a sequence Continues until condition is false
Simpler syntax for counting More flexible for complex conditions

Functions

Functions are reusable blocks of code that perform specific tasks. They help organize code and reduce repetition.

Function Examples


# Simple procedure with no parameters
PROCEDURE SAY_HELLO() {
    DISPLAY("Hello, World!")
}

# Procedure with parameters
PROCEDURE GREET(name) {
    DISPLAY("Hello, " + name + "!")
}

# Procedure with return value
PROCEDURE ADD_NUMBERS(a, b) {
    result <-- a + b
    RETURN result
}

# Procedure with multiple parameters
PROCEDURE CALCULATE_GRADE(scores) {
    total <-- 0
    i <-- 1
    REPEAT LENGTH(scores) TIMES {
        total <-- total + scores[i]
        i <-- i + 1
    }
    average <-- total / LENGTH(scores)
    
    IF average >= 90 {
        RETURN "A"
    } ELSE IF average >= 80 {
        RETURN "B"
    } ELSE {
        RETURN "C"
    }
}

Function Components

  • Name - Identifies the function
  • Parameters - Input values (optional)
  • Body - Code to be executed
  • Return value - Output (optional)

Working with Lists

Lists are ordered collections that can store multiple items of any type.

List Operations


# Creating lists
numbers <-- [1, 2, 3, 4, 5]
mixed <-- [1, "hello", 1, 3.14]  # TRUE is 1

# Accessing elements (note: Spindle uses 1-based indexing)
first <-- numbers[1]  # Gets 1
last <-- numbers[LENGTH(numbers)]  # Gets 5

# Modifying lists
numbers[LENGTH(numbers) + 1] <-- 6    # Adds 6 to end
numbers[1] <-- 0  # Changes first element to 0

# List operations
length <-- LENGTH(numbers)     # Length of list

Common List Methods

Method Description Example
append() Add item to end list.append(item)
insert() Add item at position list.insert(index, item)
remove() Remove first occurrence list.remove(item)
pop() Remove and return item list.pop(index)
sort() Sort list in place list.sort()

Common Programming Patterns

Counter Pattern


# Counting specific items
numbers = [1, 2, 2, 3, 2, 4, 5]
count = 0
for num in numbers:
    if num == 2:
        count += 1
print(f"Number 2 appears {count} times")

Accumulator Pattern


# Calculating sum of even numbers
numbers = [1, 2, 3, 4, 5, 6]
sum_even = 0
for num in numbers:
    if num % 2 == 0:
        sum_even += num
print(f"Sum of even numbers: {sum_even}")

Search Pattern


# Finding an element
names = ["Alice", "Bob", "Charlie"]
search_name = "Bob"
found = False
for name in names:
    if name == search_name:
        found = True
        break
print(f"Name {search_name} {'found' if found else 'not found'}")

Debugging Techniques

Common Debugging Strategies

  • Print statements to track variable values
  • Check for off-by-one errors in loops
  • Verify correct data types are being used
  • Test edge cases (empty lists, zero, negative numbers)
  • Use meaningful variable names

Debugging Example:


PROCEDURE CALCULATE_AVERAGE(scores) {
    # Debug print
    DISPLAY("Input scores: ")
    DISPLAY(scores)
    
    IF LENGTH(scores) == 0 {  # Check for empty list
        DISPLAY("Warning: Empty score list")
        RETURN 0
    }
    
    total <-- 0
    i <-- 1
    REPEAT LENGTH(scores) TIMES {
        total <-- total + scores[i]
        i <-- i + 1
    }
    DISPLAY("Sum of scores: ")
    DISPLAY(total)
    
    count <-- LENGTH(scores)
    DISPLAY("Number of scores: ")
    DISPLAY(count)
    
    average <-- total / count
    DISPLAY("Calculated average: ")
    DISPLAY(average)
    
    RETURN average
}

Practice Exercises

Exercise 1: Temperature Converter

Write a function that converts Celsius to Fahrenheit.


PROCEDURE CELSIUS_TO_FAHRENHEIT(celsius) {
    RETURN (celsius * 9/5) + 32
}

# Test the procedure
temperatures <-- [0, 100, 37]
i <-- 1
REPEAT LENGTH(temperatures) TIMES {
    temp <-- temperatures[i]
    fahrenheit <-- CELSIUS_TO_FAHRENHEIT(temp)
    DISPLAY(temp + "°C = " + fahrenheit + "°F")
    i <-- i + 1
}

Exercise 2: List Processing

Write a function that finds the average of numbers in a list.


PROCEDURE CALCULATE_AVERAGE(numbers) {
    IF LENGTH(numbers) == 0 {
        RETURN 0
    }
    
    total <-- 0
    i <-- 1
    REPEAT LENGTH(numbers) TIMES {
        total <-- total + numbers[i]
        i <-- i + 1
    }
    RETURN total / LENGTH(numbers)
}

# Test the procedure
test_numbers <-- [1, 2, 3, 4, 5]
average <-- CALCULATE_AVERAGE(test_numbers)
DISPLAY("Average: ")
DISPLAY(average)

Exercise 3: String Manipulation

Write a function that counts vowels in a string.


PROCEDURE COUNT_VOWELS(text) {
    vowels <-- "aeiouAEIOU"
    count <-- 0
    i <-- 1
    REPEAT LENGTH(text) TIMES {
        char <-- text[i]
        j <-- 1
        REPEAT LENGTH(vowels) TIMES {
            IF char == vowels[j] {
                count <-- count + 1
            }
            j <-- j + 1
        }
        i <-- i + 1
    }
    RETURN count
}

# Test the procedure
text <-- "Hello World"
vowel_count <-- COUNT_VOWELS(text)
DISPLAY("Number of vowels: ")
DISPLAY(vowel_count)
