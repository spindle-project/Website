Install & Run Spindle Natively

Run Spindle directly on your computer for the most flexible and powerful experience. Follow these steps to get started:

Native Method (Running on Desktop)

1 download
Download Spindle & Python
Go to our Releases page and download Spindle.py and shell.py.
Make sure you have Python installed on your computer.
2 create_new_folder
Create a Project Folder
Make a new folder anywhere on your computer (name it anything you like) to keep your Spindle files organized.
3 upload_file
Add Spindle Files
Move Spindle.py and shell.py into your new folder. This keeps everything in one place for easy access.
4 edit
Write Your Spindle Code
Create a new text file ending in .spdl (for example, my_first_program.spdl) and write your Spindle code using any text editor you like.
5 play_arrow
Run Your Code
Simple Method:
  • Double-click shell.py or open it with Python.
  • Type RUN("[FILENAME].spdl") (replace [FILENAME] with your code file).
Advanced Method:
  • Open a terminal or command prompt.
  • Use cd to go to your folder.
  • Run: python shell.py RUN("[FILENAME].spdl")
