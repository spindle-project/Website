What is the Create Task?

The Create Task is a performance task where you develop a computer program of your choice. It's worth 30% of your AP CSP exam score and demonstrates your ability to create computational artifacts.

Pro Tips

lightbulb

Start Early

Begin your Create Task at least 2-3 weeks before the deadline to allow time for revisions and troubleshooting.

record_voice_over

Record Your Video

Practice your program demonstration before recording. Speak clearly and explain what you're doing as you run the program.

edit

Write Clearly

Use specific examples from your code when writing responses. Reference line numbers and explain your reasoning.

backup

Backup Everything

Save multiple copies of your work in different locations. Don't rely on just one device or cloud service.

Test Thoroughly

Test your program with different inputs and edge cases. Make sure all required programming elements work correctly.

Document Process

Keep notes of your development process, challenges faced, and how you solved them. This will help with written responses.
