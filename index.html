Meet Spindle, the ultimate AP CSP Tool

Spindle is a brand-new open-source programming language designed to mimic the language used for the AP Test

What makes Spindle Special?

bolt
Actually on the Test

Spindle code mirrors the syntax of the AP CSP exam, so you'll be tested on what you know directly.

school
Interactive Learning

Spindle is designed for hands-on learning. You make your own code and get to see the results firsthand.

hourglass_top
Easy to Use

Spindle only has what is on the AP EXAM, no more, no less. In addition, Spindle is well documented.

rocket_launch
Quick Set Up

Spindle can be downloaded in just a few steps, or if you don't want to download anything, you can run Spindle code on our website!

We take AP CSP Seriously

Experience the power of Spindle firsthand with our interactive demo. Write code, see the results instantly, and learn the basics of the language in a fun and engaging way.

How can I use it?
Spindle is available everywhere! If you want to get started as quickly as possible, click the "code" option in the nav bar. Or if you want to download it natively on desktop, click the "download" option and follow the instructions.
