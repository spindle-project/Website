Spindle is a free, open-source language empowering students to master AP CSP exam coding with ease.

Set Yourself Up for a 5

Spindle is designed to closely resemble the syntax and structure of the programming language used in the AP Computer Science Principles (CSP) exam. This means that Spindle code looks and behaves very similarly to the code that you’ll be expected to parse on the exam!

Given that Spindle is entirely free, there's absolutely no reason not to give it a try!
